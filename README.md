# StudyNotion - EdTech Platform

An EdTech platform built for educators and contributors, designed to streamline the educational process.

## Overview

StudyNotion allows users to:
- Access a variety of educational materials
- Collaborate on projects
- Interact with educators to enhance the learning experience

## Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB

## Architecture

Monolithic architecture with a unified codebase handling both frontend and backend.
